# SametSisartenep's dotfiles

My dotfiles -- zsh . vim . rbenv . nvm

## usage

1. Clone the repo
  - `~$ git clone --recursive https://github.com/SametSisartenep/dotfiles.git`
2. Install
  - `~$ cd dotfiles && ./install.sh`

## setup LLVM + Clang

`~$ ./clang/setup.sh`

## setup V8 + Google Depot Tools

`~$ ./v8/setup.sh`
